Team Activity: Discussion

Q1. What factors determine whether a programming language is secure or not?

The various factors, according to Cifuentes and Biermann, that determine whether a programming language is secure are 
  • Buffer Errors – This occurs when the data stored overflows the designated size of the data type.
  • Injection Errors – Vulnerabilities such as Cross side Scipting (XSS), SQL Injections, etc that can compromise the security of the data or the application causes Injection Errors.
  • Information Leak Errors – Information being leaked through log files, caching, environment variables, test code, etc causes such errors.

Although a lot of such errors are mitigated by the Software Developer at the time of development of software. However, it is upto the programming language to provide the support to handle such errors to prevent the attackers to exploit the common vulnerabilities.

Q2. Could Python be classed as a secure language? Justify your answer.

Python, with the advancement of version 3.x, became a secure language. The common issues such as Overflow errors have been handled pretty well. Python 3.x raises the Overflow Error when the size of the value exceeds the possible data type. Python based web frameworks such as Flask provides basic escape character (pipe symbol |) to mitigate the DoS and XSS attacks. It also provides an excellent set of threading primitives to handle race conditions. The newer string formating function (format()) provides safer ways. Thus it can be easily concluded that Python 3.x is a secure language.

Q3. Python would be a better language to create operating systems than C. Discuss.

Python is not the language to develop the operating system. For an operating system, the language should be able to take control of memory allocation and freeing, and able to jump to specific address and continue execution from that address. Python is a high level language and is dependent on low level language such as C to directly access hardware and perform low-level data structure manipulation. However, if low level stuff is written in C and other functions in Python, an Operating system can be created.
