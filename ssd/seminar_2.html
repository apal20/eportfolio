Seminar 2

On reading the given Python Tutorial on Cryptography, I came to know a number of encryption and decryption algorithms such as Caesar, ROT13, Transposition, Base64 Encoding of files, XOR Process, Simple Substitution Cipher and Hashing alogrithms from Cryptography library.

For this seminar, I choose to use the SHA256 hashing algorithm for the Text Encryption. This algorithm provides the following benefits -

  1. It is used to store passwords in the applications. Such a use is widespread and will be utilised in our final output.
  2. This algorithm, along with SHA512, and Bcrypt are one of the most secure algorithms to save passwords.

With respect to GDPR, if we store the hash of Consumer’s Id or its Name and other identifiers, before transferring the data, it involves a level of anonymity for the third party users. Thus the identity of the consumer is protected with the help of one way hashing.

For the file encryption, I chose the PyCrypto library of Python. This encryption involves the use of a password to encrypt the file before it can be transmitted over a communication channel. Thus, if one is to tranfer any file, under GDPR regulations, then it becomes mandatory to encrypt the personal data files such as copies of identity cards.
