Seminar 1 - STRIDE and DREAD tools

Kali Linux – ePortfolio Activity

Q. What does the article teach you about carrying out vulnerability scans using Kali?

A. The given articles [1][2] tells us that Kali is an open source linux distro which provides a pre-packaged tools to perform penetration testings and vulnerability assessments. Kali comes with a specialised menu which group the applications as per the following heads -

1. Information Gathering

2. Vulnerability Analysis

3. Web Application Analysis

4. Database Assessment

5. Passwork Attacks

6. Wireless Attacks

7. Reverse Engineering

8. Exploitation Tools

With the help of Kali linux, we can sit silently on a network and perform an offensive attack on the servers, networks and application in the given environment.

Q. What issues might you encounter? 

A. While using Kali Linux, it is important that the quieter you become, the more you are able to hear. Thus it is always to keep a note not to start any services that can make the presence of the tester available to a possible attacker in a hostile network.

Q. How would you overcome them?

A. These issues can be overcome by either restarting the linux (as on restarting, Kali does not start any network services) or by manually closing all the services.

Q. What do you think of their criteria? 

A. While most of the criteria taken by them matches ours, however their criteria does not include the concept of the Privacy. For the Pen test tools evaluation, it is highly important to maintain the quiteness in the network. However, it takes into account the support or documentation levels of the tools, so as to bring a supporting factor on how easily it can be used by a newbie or to provide the complete range of commands that can be used by the given tool.

The answers given by them only contain Yes / No and does not take into account the levels on a Likert scale, which is a defacto standard for evaluation or assessments.

Q. What are the pros and cons of using Kali Linux vs. Nessus? 

A. Kali Linux is an Operating System Distribution and Nessus is a standalone application. Kali contains several applications for pen testing including Nessus. Kali Linux provides a wide range of tools to perform various sorts of attacks, pen testing or vulnerability assessment however Nessus provides only a limited set of critera to evaluate. Nessus scores huge points in the ease of use as it is mostly configure and click action, however, Kali linux requires a lot of efforts and knowledge to perform testings.
